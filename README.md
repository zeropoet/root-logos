# Root Logos

Root Logos is a living archive of constitutional writings for human-machine civilization.

## Structure

- `index.html` is the public archive page.
- `statements/` contains the public article pages.
- `styles.css` holds the visual system.
- `script.js` highlights active archive sections while reading.
- `article.js` renders canonical Markdown into the article pages.
- `content/statements.json` is the statement index metadata.
- `content/living-statement-001.md` through `content/living-statement-003.md` reserve canonical text slots.
- `assets/` contains the mark and generated archive image.

## Publishing

This repository is ready for GitHub Pages. The `CNAME` file points the site to `rootlogos.com`.
